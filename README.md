# Personal Portfolio

Frontend dasturchi sifatida yaratgan shaxsiy portfolio sahifam. Bu loyiha
mening ko'nikmalarim, qiziqishlarim va aloqa ma'lumotlarimni bir joyda
namoyish etadi.

## 🔗 Jonli demo

[Loyiha uchun havola](https://deluxe-gecko-ebf08e.netlify.app/)

> Demo linkini o'zingiznikiga almashtiring (Vercel, Netlify, GitHub Pages va boshq.)

## ✨ Asosiy imkoniyatlar

- Sof HTML, CSS va JavaScript bilan yaratilgan (framework siz)
- To'liq responsive dizayn — desktop, tablet va mobil qurilmalarda ishlaydi
- Semantic HTML va ARIA atributlari bilan accessibility-friendly
- Mini interaktiv loyihalar:
  - Counter (oshirish/kamaytirish/reset)
  - Character counter — real-time belgi sanog'i
- Aloqa formasi Telegram bot orqali xabar yuboradi
- Smooth animatsiyalar va hover effektlar
- BEM uslubidagi class nomlash konvensiyasi

## 🛠 Texnologiyalar

- **HTML5** — semantic markup
- **CSS3** — Custom Properties, Grid, Flexbox, animations, media queries
- **JavaScript (Vanilla)** — DOM manipulation
- **Google Fonts** — Plus Jakarta Sans
- **Git & GitHub** — versiya nazorati
